Example with next prev nav

A snapper carousel with next/prev links automatically added through the addition of a data-snapper-nextprev attribute. In this demo, the scrollbar is cropped from sight using the optional snapper_pane_crop div (only recommended when thumbnails or next/prev navigation is in play)

This example also has a regular thumbnail links nav with links styled instead to appear as dots.

Also, this demo includes an optional ".snapper_nextprev_contain" div, which the next/prev nav will append to if present. This element wraps the scroll pane and allows you to precisely position arrows based on the height of the pane, without the thumbnails or dot nav.
